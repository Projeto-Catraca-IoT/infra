# Kubernetes

Para rodar a aplicação, siga esses passos simples:

**Observação:** Vamos considerar que você possua o Kind e o Cubectl instalados

1. Faça o download dos arquivos [dessa](https://github.com/Projeto-Catraca-IoT/infra/docker/compose.yml) pasta em sua máquina
2. No mesmo local que você os baixou, execute esse comando:
``` bash
    chmod +x script.sh
```
3. Rode o seguinte comando em sua máquina:
``` bash
    #Certifique-se de estar no mesmo nível do arquivo script.sh
    ./script.sh
```
4. Agora basta acessar
